# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-18

### Added
- Initial release of rkk-next SDK
- SEO optimization components (MetaManager, JsonLd)
- Smart routing with intelligent prefetching (SmartLink, prefetchRoute)
- Performance optimization utilities (OptimizedImage, lazyImport)
- Cache header presets (LONG_TERM_CACHE, EDGE_CACHE, etc.)
- Security headers configuration
- Web Vitals reporting integration
- Route observer for analytics
- Full TypeScript support
- Support for both Pages Router and App Router
- Comprehensive documentation and examples

### Features
- **SEO**: Meta tags, OpenGraph, Twitter Cards, JSON-LD schema
- **Routing**: Hover-based prefetching, route change observer
- **Performance**: Lazy loading, image optimization, CDN caching
- **Analytics**: Web Vitals tracking, route analytics
- **Security**: XSS protection headers, content security
